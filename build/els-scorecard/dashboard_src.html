Read this first. This is an experimental analytical product, not official statistics — values are quoted from published official statistics, but ranks, quartiles and better/worse directions are analytical judgements; treat with caution and verify against source releases before using in decisions (assumptions and limitations: analytical assurance note). Indicators come from many producers with different reference periods (noted on every row) — this is a compendium, not a single snapshot. Survey-based indicators (marked ≈) carry 95% confidence intervals; for a borough of Gateshead's size these are wide, so treat their ranks as soft. Money values are current prices unless marked otherwise. Ranks compare Gateshead only with local authorities that have data; context indicators (population size, house prices, traffic) are shown without a better/worse reading.
