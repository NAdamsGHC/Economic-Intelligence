Read this first. Indicators come from many producers with different reference periods (noted on every row) — this is a compendium, not a single snapshot. Survey-based indicators (marked ≈) carry 95% confidence intervals; for a borough of Gateshead's size these are wide, so treat their ranks as soft. Money values are current prices unless marked otherwise. Ranks compare Gateshead only with local authorities that have data, and "better/worse" follows the stated direction of each indicator; context indicators (population size, house prices, traffic) are shown without a better/worse reading.
